# 电影推荐系统简介 python656

## 项目概述

本项目是一款基于 Python 的协同过滤算法的电影推荐系统。系统维护了爬虫功能，并根据参考论文持续优化推荐算法，致力于为用户提供个性化的电影推荐服务。

## 技术栈

- **后端语言**: Python
- **开发框架**: Django
- **开发工具**: PyCharm
- **数据库**: SQLite

## 功能模块

### 前台管理

- 电影信息展示
- 电影资讯浏览
- 电影搜索
- 电影收藏
- 用户评分

### 后台管理

- 电影分类管理
- 用户管理
- 电影评分管理
- 系统管理

## 系统角色

- 普通用户：可浏览电影信息、收藏电影、对电影进行评分等。
- 管理员：负责电影分类、用户管理以及系统设置等。

## 运行环境

- Python 3.x
- Django 2.x
- SQLite 3.x

## 部署步骤

1. 确保Python环境安装，并安装必要的依赖包。
2. 下载源码并安装SQLite数据库。
3. 使用PyCharm或其他IDE打开项目，并配置数据库连接。
4. 运行爬虫功能以初始化电影数据。
5. 启动项目，通过浏览器访问前台和后台界面。
